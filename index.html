Gazetteer


In this project I learnt how to use API's to fetch real-time data from servers in JSON format and present it organized.

Company directory


This is a CRUD like app designed to manage the internal organisation of a company. Allowing user to Create, Read, Update and Delete information on the database

Earthquake Mapper


This program in JavaScript and PHP fetches data in real time from the US Geological Survey API and represent every Earthquake happened in the last week. I included this project as an extra functionality in my Gazetteer.

ISS Tracker


This project works similarly to the Earthquake Mapper. Built in JavaScript and PHP. It gets the exact position of the International Space Station and represents it in the map. Also included in my Gazetteer as an extra functionality.

To-Do App


Simple To-Do list app with standard bootstrap style and modular php back-end for easier maintenance. You have your daily tasks in a list and you can add, edit and delete tasks without refreshing the page.

Login & Registration System


Simple Login & Registration system in Javascript, PHP and MySQL. Features password recovery.
